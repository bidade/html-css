Listas ordenadas

As listas ordenadas são aquelas que possuem uma ordem lógica como uma receita, para que tenhamos que fazer as ações uma após a outra como idicado na lista, veja abaixo o exemplo de uma lista ordenada:

  1. Ir ao mercado
  2. Pegar a cestinha
  3. ir até as seções desejadas
  4. Pegar os produtos de sua preferência
  5. Ir ao caixa
  6. Passar os produtos
  7. Pagar o valor total dos produtos
  8. Aguardar a moça colocar os produtos em uma sacola
  9. Pegar os produtos e sair do mercado

Listas não ordenadas

As listas não ordenadas, como já diz não precisa de uma ordem de execução. veja abaixo

  • Pão de queijo
  • Sabão em pó
  • Sacola plastica
  • Arroz

Tipos nas tags ul e ol

Nas listas ordenadas(ol) por padrão já vem com ordem numerica, mas podemos alterar o type para (Alfabetico maiusculo e minusculo, ou algarismo romano, maiusculo ou minusculo) também podemos indicar em qual posição irá se iniciar a lista (podendo começar do c, 3, XI) com o parametro 'start="posição"'Veja abaixo alguns exemplos:

lista com o tipo Alfabetico maiusculo

  1. Arroz
  2. Abacaxi
  3. Pipoca
  4. Couve

lista com o tipo alfabetico minusculo começado a partir do c

  1. Arroz
  2. Abacaxi
  3. pimenta

Mudando o tipo da lista não ordenada

Como podemos ver, as listas não ordenadas por padrão começam com uma bolinha preta, saiba que também podemos mudar isto usando o paramentro style (circle, square e disc).

Veja abaixo o exemplo: <ul type='square'>

  • Ovo
  • Arroz
  • Peixe

podemos notar que ficou o quadrado preto antes do item, isso porque o type square deixou desta forma

Aninhando listas e sub listas

Podemos colocar listas dentro de outras listas, criando assim uma sub lista. não só lista do mesmo tipo, podemos colocar uma lista ordenada com uma sub lista não ordenada.

Lista de definições

Há também um tipo de lista para nos referirmos a termo e definição veja abaixo:

Esse metodo consiste em um termo e uma definição

HTML
Curso onde veremos como estruturar um site
CSS
Curso onde aprenderemos a "embelezar" o site
Javascript
Curso onde aplicaremos as funcionalidades ao nosso site

Como você pode ver acima, para cada termo nós indicamos uma definição, assim que funciona a lista de definição com as tagas dl(abertura) td(abertura do termo) dd(abertura do significado ) *as 3 possuem fechamento*
